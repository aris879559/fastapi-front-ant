# fastapi-front-ant
fastapi-front-ant 后台前端react项目，基于Ant Design Pro实现

## 项目启动方式
```shell
    git clone git@github.com:aris879559/fastapi-front-ant.git
    yarn
    # Windows上执行 
    set NODE_OPTIONS=--openssl-legacy-provider
    #Linux上执行
    export NODE_OPTIONS=--openssl-legacy-provider
    yarn start
```
